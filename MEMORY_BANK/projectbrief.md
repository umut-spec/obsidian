# Obsidian - Project Brief

## Proje Adı
**Obsidian** — Performance & Security focused Paper Fork

## Vizyon
Minecraft sunucu yöneticileri için en güvenli ve performanslı Paper fork'u oluşturmak. Pufferfish/Purpur seviyesinde optimizasyonlar + kapsamlı güvenlik korumaları.

## Temel Bilgiler
- **Base:** PaperMC/Paper v26.1.2 (MC 26.1.2)
- **Java:** JDK 25+ (Adoptium Temurin 25.0.3+9)
- **Build System:** Gradle 9.4.1 + paperweight-core 2.0.0-SNAPSHOT
- **Package:** `dev.obsidianmc.obsidian`
- **Versiyon:** 1.0.0

## Temel Hedefler
1. **Performans** — Entity, hopper, TNT, chunk, redstone optimizasyonları
2. **Güvenlik** — Packet limiter, anti-bot, exploit korumaları
3. **Konfigürasyon** — Tek dosya (`obsidian.yml`) ile tüm ayarların yönetimi
4. **Uyumluluk** — Paper plugin API ile %100 uyumlu
